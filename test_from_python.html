Section 1: Lee's Notes So Far (Stage 2 - Work Session 1)

World Wide Web

The World Wide Web is a collection of computers that can communicate with each other to share html and source files for other computers on the WWW to view.

Major Pieces of the Web

  • You
  • Your Computer + Browser
  • The Internet
  • Servers

HTML is

  • Text content — What you see
  • Markup — Code to display
  • References to other documents — e.g. images & videos
  • Links to other pages

Parts of HTML

  • Tags - Opened and closed with < & > respectively.
  • Elements - Everything from an opening tag to it's closing tag, with everything in between.

Inline & Block Elements

Some elements can be "inline" which meants they would go along in a sentance as if it was a word. Others can be "block" which means that the computer sees it as having its own space and breaks out of the line to put a box around it.

HTML Code Structure

HTML code must be structured for ease of use and accuracy in editing. Without your code structured it is very hard to make your web page look like anything but a basic word document. With structure from the apropriate elements such as DIV tags and such you can then manipulate these with CSS to look and act like anything you want.

Avoid Repetition

Repeating styling codes not just wastes space and creates bigger, bulkier files but it also makes it harder to change document-wide styles and structure. Starting at top level selectors like classes and pseudo tags can make it easy to quickly change the whole appearance of a web page. For more precise and individual formatting selectors such as IDs and attributes can be used which, as the are more specific selectors, will only edit the styles for the specific tags even if a higher selector is modifying them.

Summary for Lessons 1 & 2

All HTML is broken down into elements that are made up of tags, attributes and content.

Formatting the elements in your code to display the HTML and CSS will let you make just about anything visible to the viewer of your web page.

Summary for Lesson 3 (Big Ideas)

Using CSS you can get all the boxes and inline text to act as you want. You do this by using selectors. Selectors can be tags, classes, ids, attributes, seudo tags and pseudo elements. You then assign properties (style) to those selectors and anything that has the selector will be styalized as defined.

To get the desired result of a web page that looks like a design you go through a cycle of CODE → TEST → REFINE until you get the document laid out the way you want. You start this by;

  1. Looking for the natural boxes of the design.
  2. Look for repeat styles and syntax elements so that these can be given grouped classes or other selectors.
  3. (CODE) Write the HTML.
  4. (CODE) Write the styles in CSS starting from the more broad aspects and then going down to the smaller details.
  5. (TEST) Test and (REFINE) fix the things that are not correct by repeating steps 3 & 4.

Variables

Variables are used in most programming. The are something that is defined for the current program. This action is called "assigning a variable". It usually consists of a name and then an exuals sign, and then the value, text or other variable, or combination thereof, being assigned that name. i.e. name = 'Lee Gellie' or sum = 4 + 2

In the Python program, and most programming, the = in a variable is not just 'equals', it means 'assignment'. It does not mean summation!

Comments

Comments can be defined in Python code using a #. When doing this, everything behind the # will be commented and not read in executing the code. In python each line is a command so this does not need to be closed, it ends when the line ends. It is very useful in making notes so you, or others, can recognize what the code is doing or why you went about it a certain way.

Strings

Strings are generally text values. The will be defined with a start and an end quote. Such as 'Lee' or "Lee". The quotations that you start and end with must be the same. You can not start with a single and end with a double. If you do not end a quote it will be an error.

This makes it very handy for using the other kind in the text as EVERYTHING will be concidered text until you have closed the quotation. So you can write print "Lee's mum is going to the 'new' house today!". This would return a value of Lee's mum is going to the 'new' house today!

Python can not add strings and numbers. print "Joe" + 39 would be an error. If you wanted to add text and a number you could do this by assigning a variable such as;

age = 38

Selecting Sub–Sequences

If you add opening and closing square brackets after a variable calling a string you can use this to numerically select parts of that string.

All string selecting starts with '0' and goes up numerically to however long the string is. You can select by numbering from the back of the line starting at -1 and work your way toward the beginning. So to select the second character of the variable string = plop you would need to type string[1] and the result would be 'l'.

You can also select multiple characters from the string by using 3 spaces in the square bracket represented as Number–Colon–Number, or Start–Colon–Stop. You do not, in fact, have to have a number before or after the colon but this is the easiest way to think of it. If you have nothing before the colon it will start from the beginning of the string. If you put a numerical value it will start from that column of the string. The colon means 'go up to'. And then you have the final number space, which will take everything from the beginning or the defined start and go up to JUST BEFORE the final number. So if the last character was 23 and you typed string[1:4], you would get from the 2nd, remember it starts on 0, to the second last charcter. So, in the above example you would get returned lo. Again, if you leave the end number blank it will go to the end. If you typed string[1:] you would get returned lop. So if you left both numbers blank string[:] you would get from the beginning to the end, so plop.

Using the Find Operation

The find command can be used in the following format string_var.find(string)

It will search from the beginning of the string until it finds the first instance of the search parameter. It will then return the numerical column that the first part of the parameter was found. If it does not find it at all it will return -1.

You can define where the search starts by typing string_var.find(string, 14) which will have the search start on the 15th character in the string.

Section 4: Stage 2 — Input→Function→Output

Concepts of Input→Functions→Output

In the most simplistic terms you enter something into a computer, calculator, toaster or whatever and it does something in some way to give an output. In Python you define things and give it data, which can be strings or numbers, and have the program in some way use the code to produce something which will then be returned or given back as a result. Every line after the defining of the function name must be indented.

For example if I wrote;

def say_hello(name):
              greeting = "Hello " + name + "!"
              return greeting

I would get the result;

print say_hello("Miriam")
print say_hello("Andy")

The "Mirriam" & "Andy" would be input. The code to create the statement would be the function and the result and what were displayed would be the output.

Using a Procedure (Function)

You can set (make) procedures by using a def command at the beginning and it won't use it unless called upon later in the code. It will be there and be dormant until you set a command to run that procedure and then return it's value.

To set a procedure the basic code is to give it a name, for example jojo, then you have opening and closing parentheses with any input value you want to pull in them separated by commas. So something like jojo(a,b)

There can be as many inputs as you want as long as you separate them with commas and give that many inputs.

If you are defining a procedure you put def before it and a colon after and then on the next line you put the function(s) you would like run.

You end up with something like this;

def add_two_numbers(number_1, number_2):
                      return number_1 + number_2

You could then give the command along with the input;

print add_two_numbers(4, 3)

And you would get the result 7.

An important thing to remember is that you must have an output for your function. You can do all the clever stuff you want but without a return your print statement will still be blank.

Section 5: Stage 2 — IF & WHILE

Making Decisions with Code

The operators for making decisions in Python give a boolean (true or false) result. Some of the things we looked at are greater than (>), less than (<), equal to (==) and not equal to (!=). The double equals symbol is used as the single equal symbol is already used to assign variables.

Using the IF Function

To use the if function type if <expression>: then below and indented you have the <block> which will define what code is run if the expressions is TRUE. If the expression is false it will skip this code.

With the IF function you can also define an ELSE statement. This means that if your IF runs a false result it will then go to the else statement and run what is there. The ELSE must be on the same indentation as the IF, as in the example below;

def bigger(num_1,num_2):
    if num_1 > num_2:
        return num_1
    else:
        if num_1 < num_2:
            return num_2
        else:
            if num_1 == num_2:
                return num_1

This code will always return the greater of two numbers input.

If you are telling the code to return True or False you must capitalize as below;

def is_friend(name):
    if name[0] == 'D':
        return True
    else:
        return False

In the above example you can simplify it by just asking for the value as the double equals sign is already testing true or false. You could then have something like this to get the same result;

def is_friend(name):
    return name[0] == 'D'

Some people like to have all the statements but it is not necessary.

You could also give an or statement if you want to check against several values being True;

def is_friend(name):
    return name[0] == 'D' or name[0] == 'N'

Evaluating IF and OR

When a function, such as IF or OR is called to evaluate two or more expressions it will stop at the first TRUE expression. So if you said Am I 8 or 9? and you were 8, it would return True. It would never actually evaluate whether you were 9 or not as it knows at least one of the statements is true already. If you were 9 and asked the same question it would evaluate the 8 as FALSE but then go onto the 9 and return TRUE.

So if you needed to test both expressions you may need to just put False in the first to ensure that there is no error in the second, as it will not pass the first expression if it returns a TRUE value. It would then have to evaluate that second expression and tell you if there was an error in it.

The MAX command

Working out how to get the biggest number from 3 different numbers can easily be done with the MAX command which will automatically give you the largest of any amount of inputs. You would write it;

max(a,b,c,etc...)

Loops

A loop is basically a way of repeating a certain action. We are starting with the WHILE loop.

The WHILE loop starts with an expression that will validate the running of the loop followed by a colon. This is called a test expression. Inside of that we have the block that will run as long as the test expression is TRUE. The difference between this and the IF command is that once it runs the command it will go back to check if the test expression is still TRUE. If it is, it will run the block of code again, and it will keep doing that until the test expression shows a FALSE value.

i = 0
while i < 10:
    print i
    i = i + 1

This will result in the code printing from 0 to 9. Once it gets to 10 it is no longer less than 10 so the code will not work.

An Infinit Loop is one that never finds a false value so will keeo running indefinitely.

Break Statement

A break gives you a way to stop the loop even if the condition is still TRUE.

You could stick this in an IF command in the WHILE loop which will check to see if there is a reason to break the process or let it continue. If this IF statement is TRUE and you break out of the loop you will then continue the code after the while loop. If it the IF statement is FALSE it will continue to run the code and the loop and will check the IF statement again next time around.

Summary for Stage 2 – If & While Functions

The IF and WHILE functions can be extremely useful and make life a lot easier for a programmer as you can automate when and for how long (or how many times) code steps get carried out. Without them you would have to write code for every possible occurrence and it could be very lengthly and extensive to code.

From counting to greeting according to time of day, these functions can make it very simple to code aspects that would otherwise take a lot of consideration and figuring.

Section 6: Stage 2 — Structured Data

List Function

It is similar to a string, in that it is a type of structured data. But a string is limited as it is strictly text data.

In a list you can have any kind of data. In a string instead of using quotation marks you use square brackets and you divide the data with comments, like so; [a,b,'yoy','flop',4].

If you had the string;

s = [a,b,'yoy','flop',4]

You can still use the square brackets to call data from it like;

s[3]

Return would be 'flop'.

If you selected multiple values, as;

s[3:4]

You would get returned a list and the return would be;

['flop',4]

You can make a list with no elements [], known as an 'empty list' or any number from one element upwards. Like in strings, the first element is referred to as 0.

You can also have a list inside of a list, in which case you just add a set of square brackets in between the commas of you original list and populate it as you would a normal list.

If you have to divide a list onto more than one like you should try and do it after a comma in the list.

If you need to get an element out of a list inside a list you would just ad a second square bracket after the first.

a[2][3]

This would give you the 4th nested list item in the 3rd list item of the main list.

Mutation

Mutation means we can change the value of a list after it has been created. So if you had a list p = ['H','e','l','l','o'].

You could replace the 'H' to 'Y' by redefining the first element of the list like this: p[0] = 'Y'

Strings are not able to be changed. Lists have a big advantage there.

Aliasing

Is when you have 2 different names for the same object. If you change one then the other will also change.

Append Function

Like find you have <list> .append(<element>) This will affix an element to the end of a list. It will not make the list different. This will add whatever is in the append into it's own list item. if you wanted to add several items into the list you would have to use the PLUS function as the append would make a new list inside of the original list as one item on the end. Basically it means add one list item containing...

Plus Function

You can add a plus sign between two or more lists and it will give you an output of a list combined of all the items in each list added together.

Len Function

len(<list>) The LEN operator is short for Length. It also works for strings. If you put a list inside the length operator it will return the number of base list items in the list. If you have a list inside the list it will still only count it as one item.

If you use the length operator on a string it will count the number of characters in the string.

Append, Plus and Len

If you had

p = [1,2]

q = [3,4]

And appended them: p.append(q)

You would get len(p) = 3 as the list would have appended q as one list item. So p would now refer to: [1,2,[3,4]] Which is three base list items.

If you wanted to add them you would use the plus, like so;

p = [1,2]

q = [3,4]

p = p + q

len(p) now = 4, [1,2,3,4]
